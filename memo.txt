

技術選定

- フロントエンド
  Language: TypeScript5.0
  Library: React18
  FW: Next.js13.5(App routerを採用)

- CMS
  「Contents Management System：コンテンツ・マネジメント・システム」
   Webサイトのコンテンツを構成するテキストや画像、デザイン・レイアウト情報（テンプレート）などを一元的に保存・管理するシステムのこと.
   めちゃくちゃ簡単に言うと、データを保存しておく、DBの処理などのバックエンド側の処理をまとめてやってくれるシステム
  microCMS
  誰でも使いやすいことを目標にして開発された日本製ヘッドレスCMS
  クレジットカードなしで使える.

- CSS
  TailwindCSS
  TailwindCSSは、ユーティリティクラス(utility class)を自由に組み合わせて活用することで、
  オリジナリティの高いデザインのWebサイト及びWebアプリケーションを作成可能なCSSフレームワーク.
  Next.jsのプロジェクトに導入するのがすごく簡単(コードは汚くなる)

- デプロイ
  Vercel
  VercelはCI/CDとWebサーバーが合わさったサービス.
  無料で始めることができ、GitHubなどのリポジトリと連携することで手軽にアプリケーションをデプロイすることが可能.
  CDNが含まれているため高速なページ表示が可能
  使うプランはHobbyなので収入を得るような使い方はできません。有料プランにするか、他のデプロイ先を検討.




------------ 1/8 ------------------------------
・型チェックをスクリプト化して簡単に実行できるようにするために package.jsonに以下を追加

  "scripts": {
    "type-check": "tsc --noEmit"
  }

  これにより、次回以降は以下のコマンドで型チェックを実行できる.

  npm run type-check
